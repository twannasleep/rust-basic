## 🚀 Getting Started

1. **Install Rust**
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. **Clone Repository**
   ```bash
   git clone https://github.com/yourusername/rust-tutorial.git
   cd rust-tutorial
   ```

3. **Install Tools**
   ```bash
   rustup component add rustfmt clippy
   cargo install cargo-edit cargo-watch
   ```
